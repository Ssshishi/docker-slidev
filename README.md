## CI问题总结

- not resolved   --->  没有找到 或者是 大小写出现问题了 导致没找到
- Process completed with exit code 1  -----> 下面代码有问题，语法问题，或者是其他的参数获取问题 一一排除
- 没有设置TOKEN ---设置token
- Js 文件 not found    ---路径不全，相对路径，要改成正确的全路径
- redirect --- 重定向问题，其他图片文件被重定向至第一个页面，导致图片找不到 ------ 先去除，然后想办法区分文件
